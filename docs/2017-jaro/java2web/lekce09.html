Lekce 09 - Databáze a databázový server MySQL

Soubory

Databázový server MySQL rozbalte do C:\Java-Training. Není třeba ho instalovat, jen rozbalit pomocí 7-Zipu.

MySQL Server se spouští pomocí C:\Java-Training\MySQL\startup.bat.
Zastavuje se pomocí C:\Java-Training\MySQL\shutdown.bat.

Archív najdete zde: MySQL.7z
