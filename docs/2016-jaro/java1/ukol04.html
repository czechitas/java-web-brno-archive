Lekce 04 - Rozšiřující úkol na doma

Hozené číslo se zobrazuje pouze jako text. Bylo by ale hezké zobrazit výsledek jako obrázek s puntíky. Obrázky s puntíky jsem vám dal k dispozici do C:\Java-Training\Projects\Lekce04\kostka?.png
Zobrazit jednoduchý obrázek pomocí JLabelu už umíme. Jak ale přepínat mezi obrázky se správným počtem puntíků? Jednoduše. Pomocí vizuálního návrháře vložte na okno 6 komponent JLabel. Každé nastavte jeden obrázek s puntíky. Následně je všechny zneviditelněte pomocí vlastnosti visible nastavené na false. Při hodu kostkou vyberte odpovídající JLabel a nastavte mu vlastnost visible na true. Ostatním JLabelům nastavte tuto vlastnost naopak na false. Například takto:

if (cislo == 1) {
    labHozeno1.setVisible(true);
} else {
    labHozeno1.setVisible(false);
}
