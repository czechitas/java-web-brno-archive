Kurz Javy pro mírně pokročilé

Kurz Java 2 (navazující na Java 1 z minulého semestru) je zaměřen na důkladné ovládnutí jazyka Java a schopnost samostatného vývoje jednoduchých appek pro Windows/Linux/Mac. Když už víš, co čekat od programování a jak to v praxi probíhá, budeme se moct zabývat Javou důkladně. Budeme mít víc teorie a náročnější příklady. Po absolvování bys měla být schopna už začít programovat svoje vlastní malé appky bez vedení lektorů.

Druhý cíl je naučit tě běžně používat programátorský editor IntelliJ IDEA. Tedy: Jak založit projekt v editoru IDEA; spustit tento projekt; používat našeptávání; ladit chyby ve tvém programu; jak připravit distribuční balík tvojí appky, aby si ho mohli spustit i ostatní. V tomto bodě jde vlastně o to, aby sis zvykla na běžnou programátorskou praxi.

Budeme pracovat typicky na malých hrách (Angry Birds, pexeso, a podobně). Budete mít většinu programu hotovou, bude potřeba jen dokončit chybějící kousky.

Osnova lekcí

  1. Jak běží javový program (metoda main)
  2. Přehled existujících balíků a tříd v Javě
  3. Proměnné a jejich typy
  4. Třída jako složený datový typ
  5. Metody
  6. OOP
  7. Zapouzdření, metody get a set
  8. Konstruktor
  9. Dědičnost
  10. Rozhraní
  11. Používání knihoven třetích výrobců
  12. Aplikace pro počítač (okna)
  13. Práce se soubory
